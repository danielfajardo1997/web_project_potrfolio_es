# Web Portfolio - Daniel Fajardo

## 📌 Descripción del proyecto

Este proyecto consiste en el desarrollo de una página web responsiva que presenta el perfil profesional de un desarrollador web. El objetivo es mostrar información personal, habilidades y enlaces relevantes, aplicando buenas prácticas de desarrollo frontend.

La página está diseñada para adaptarse a diferentes dispositivos, ofreciendo una experiencia de usuario clara y accesible.

---

## 🚀 Funcionalidad

* Diseño responsivo (adaptado a móvil, tablet y escritorio)
* Sección de presentación personal
* Organización clara del contenido
* Uso de estilos modernos y estructura semántica
* Navegación sencilla e intuitiva

---

## 🛠️ Tecnologías utilizadas

* HTML5 (estructura del sitio)
* CSS3 (estilos y diseño responsivo)
* Metodología de organización de archivos (carpetas separadas para estilos, imágenes y fuentes)
* Git y GitHub (control de versiones)
